Microchip Technology Inc.

File Name

ftpc.h

Description

File Transfer Protocol Client - FTPC 

The FTPC client module in the TCP/IP Stack lets applications send e-mails to any recipient worldwide. These message could include status information or important alerts. Using the e-mail to SMS gateways provided by most cell phone carriers, these messages can also be delivered directly to cell phone handsets.
