TCPIP_FTPC_DATA_EVENT_TYPE Enumeration

Enumeration: TCPIP_FTPC_DATA_EVENT_TYPE 

Enumeration describing the possible event codes for FTP client data socket.

C
typedef enum {
  TCPIP_FTPC_DATA_RCV,
  TCPIP_FTPC_DATA_RCV_DONE,
  TCPIP_FTPC_DATA_SEND_READY,
  TCPIP_FTPC_DATA_SEND_DONE
} TCPIP_FTPC_DATA_EVENT_TYPE;
Members
Members 
Description 
TCPIP_FTPC_DATA_RCV 
when data received on Data Socket 
TCPIP_FTPC_DATA_RCV_DONE 
when data receive complete 
TCPIP_FTPC_DATA_SEND_READY 
when Data Socket is ready to transmit data 
TCPIP_FTPC_DATA_SEND_DONE 
when data transmission complete 
Remarks

None
