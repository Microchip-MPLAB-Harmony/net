Microchip Technology Inc.

File Name

tcpip_manager.h

Description

Microchip TCP/IP Stack Definitions 

The TCP/IP manager provides access to the stack initialization, configuration and status. It also process the packets received from different network interfaces (MAC drivers) and dispatches them accordingly based on their type to the proper higher layer in the stack.
