Building the Application

The following table lists the projects available for this demonstration. These projects are available inside <install-dir>/net/apps/web_net_server_qspi_mpfs/firmware. 

To build this project, open the <project name>.IAR project in IAR Embedded Workbench IDE. 

 

IAR Embeded Workbench IDE Project

This table lists the name and location of the IAR IDE project folder for the demonstration. 

 

Project Name 
Target Device 
Target Development Board 
Description 
sam_a5d2_xult.IAR 
ATSAMA5D27 
SAM A5D2 Xplained Ultra 
Demonstrates the TCP/IP Web Server on a development board with the ATSAMA5D27 device. This is a bare-metal (non-RTOS) implementation 
sam_a5d2_xult_freertos.IAR 
ATSAMA5D27 
SAM A5D2 Xplained Ultra 
Demonstrates the TCP/IP Web Server on a development board with ATSAMA5D27 device. This implementation is based on FreeRTOS. 
sam_9x60_ek.IAR 
SAM 9X60 
SAM 9X60-EK 
Demonstrates the TCP/IP Web Server on development board with SAM9X60 device and an on-board KSZ8061 PHY. This is a bare-metal (non-RTOS) implementation
