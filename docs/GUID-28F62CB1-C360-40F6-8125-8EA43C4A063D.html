2.1.6 Clock Configuration

  1. Start Clock Configuration from Plugins drop-down menu
  2. Verify the clock configuration. The application demo can work with default clock configuration.
    1. Verify the Processor Clock (HCLK) is 300,000,000 Hz.
    2. Verify the Host Clock (MCK) is 150,000,000 Hz.
      Note: This setting is valid for SAME70 devices; other devices will have different clock settings. For more information, refer the device datasheet.
