1.2.1.12 HTTP Module

The HTTP Web Server module allows a development board to act as a Web server. This facilitates an easy method to view status information and control applications using any standard Web browser.

Note: The HTTP module is deprecated and it is maintained for backward compatibility only. It will be eventually removed from the TCP/IP Stack.

New designs should use the HTTP_NET module.
