2.1.2 Hardware Requirements

The hardware development boards supported by Harmony 3 TCP/IP applications are as follows:

  • SAME70 Xplained Ultra
  • SAMV71 Xplained Ultra
  • SAME54 Xplained PRO
  • SAMA5D2 Xplained Ultra
  • SAM9X60-EK Evaluation Kit
  • SAM9X75-DDR3-EB Evaluation Board
  • SAMA7G54-EK Evaluation Kit
  • PIC32MZ Embedded Connectivity with FPU (EF) Starter Kit
  • PIC32MZ Embedded Graphics with Stacked DRAM (DA) Starter Kit
  • PIC32CZ CA80 Evaluation Board
  • PIC32 Ethernet Starter Kit II

Refer to the User guide of these development boards for more details.
