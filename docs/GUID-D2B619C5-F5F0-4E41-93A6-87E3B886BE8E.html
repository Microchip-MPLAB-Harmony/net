1.2.1.12.5 HTTP_READ_STATUS Enumeration

C

typedef enum {
    HTTP_READ_OK = 0u,
    HTTP_READ_TRUNCATED,
    HTTP_READ_INCOMPLETE
} HTTP_READ_STATUS;

Description

Result states for TCPIP_HTTP_PostNameRead, TCPIP_HTTP_PostValueRead and TCPIP_HTTP_PostReadPair.

Members

|Members |Description | |HTTP_READ_OK = 0u |Read was successful. | |HTTP_READ_TRUNCATED |Buffer overflow prevented by truncating value. | |HTTP_READ_INCOMPLETE |Entire object is not yet in the buffer. Try again later. |
