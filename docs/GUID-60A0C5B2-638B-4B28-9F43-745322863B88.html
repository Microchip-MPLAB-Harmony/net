2.1 Create your first TCP/IP Application

This tutorial demonstrates how to create a TCP/IP application project with MPLAB® Harmony TCP/IP Stack and MPLAB® Harmony v3 modules, using MPLABX IDE and MPLAB® Code Configurator.

The tutorial implements a simple TCP/IP Client application on a Microchip 32-bit microcontroller development board. This will guide you through the steps to create a new TCP/IP application.

Note : This guide demonstrates the creation of a TCP/IP demo on the SAME70 Xplained Ultra board with LAN8740 PHY Daughter board. These steps can be extended for other Microchip hardware development boards supported in Harmony v3, by applying the device/board specific configuration changes.
