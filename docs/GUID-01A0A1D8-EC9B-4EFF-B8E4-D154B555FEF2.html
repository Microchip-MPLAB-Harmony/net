1.2 MPLAB® Harmony TCP/IP Library

MPLAB® Harmony TCP/IP Library includes implementation of various network protocols. Each of these modules belongs to one of the layers of TCP/IP model: Application, Presentation, Transport, Network, Datalink and Physical Layer.
