MHC Configuration

 

From the MPLAB X IDE menu select Tools->Embedded->MPLAB Harmony Configurator. 

After selecting the "MPLAB Harmony Project Path" and pressing the "Launch" button, click the "Open default saved state file" and the following MHC project graph should be displayed: 

 

 

Click on the "TCP/IP Stack" box and then select the "APPLICATION LAYER to visualize the iperf module configuration: 

 

 

Notice that very few application layer modules are selected for this demo. The focus of this demo is on running the network benchmark program without supporting additional services. 

 

The default sizes of the Iperf RX and TX buffers are set to 4KB. The performance of a socket is highly dependent on the size of its buffers so it's a good idea to use as large as possible buffers for the sockets that need high throughput. Bigger buffers will help obtain higher performance numbers. 

The other iperf parameters could be left with their default values. 

For a description of the parameters see the Iperf Module
