PIC32MZ + SDMMC PICtail

Hardware Configurations 

This configuration is for the PIC32MZ EF and PIC32MZ DA Starter Kits.

  • Refer to the PIC32MZ EF Starter Kit User Guide
  • Refer to the PIC32MZ DA Starter Kit User Guide
    • Refer to the user guide for the programming/debugging options supported by PIC32MZ DA Starter Kit
  • Refer to the Starter Kit I/O Expansion Board - DM320002 User Guide
  • Refer to the SDMMC PICtail daughter board - AC164122 User Guide
  • The picture below uses the PIC32MZ EF Starter Kit but it applies to the DA Starter Kit as well.
 

  • Insert the SDMMC PICtail daughter board (AC164122) into the SPI1 slot of the I/O Expansion Board (DM320002)
  • Insert the PIC32MZ EF/DA Starter Kit into the I/O Expansion Board (DM320002)
  • Establish a connection between the router/switch with the PIC32MZ Starter Kit through the RJ45 connector on PHY daughter board
  • Power the board using the 9V power adapter
 

The demo application is configured for the SPI1 slot, using the following I/O pins:

  • SCK1 – RD1
  • SDI1 – RD14
  • SDO1 – RD10
  • CS pin – RB1
