TCPIP_MODULE_SIGNAL_FUNC Type

TCPIP module signal function 

Pointer to a function that will be called from within the TCP/IP stack when a signal is delivered to a stack module.

C
typedef void (* TCPIP_MODULE_SIGNAL_FUNC)(TCPIP_MODULE_SIGNAL_HANDLE sigHandle, TCPIP_STACK_MODULE moduleId, TCPIP_MODULE_SIGNAL signal, uintptr_t signalParam);
Preconditions

None.

Parameters
Parameters 
Description 
sigHandle 
signal handle obtained from a registration call 
moduleId 
module that receives the signal 
signal 
the occurring signal 
Returns

None.

Remarks

This function should be kept as short as possible and it should not include blocking or polling code. It's for setting flags/signaling purposes only.
