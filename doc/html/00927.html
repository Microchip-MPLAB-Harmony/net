IPV6_TLV_OPTION_TYPE Union

Type: IPV6_TLV_OPTION_TYPE 

This type defines the data structure for IPv6 TLV options.

C
typedef union {
  unsigned char b;
  struct {
    unsigned option : 6;
    unsigned unrecognizedAction : 2;
  } bits;
} IPV6_TLV_OPTION_TYPE;
Remarks

None.
