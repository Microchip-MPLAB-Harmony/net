TCPIP_DHCP_INFO Structure

Structure: TCPIP_DHCP_INFO 

This data structure is used for reporting current info and status of the DHCP module. Used in getting info about the DHCP module.

C
typedef struct {
  TCPIP_DHCP_STATUS status;
  uint32_t dhcpTime;
  uint32_t leaseStartTime;
  uint32_t leaseDuration;
  uint32_t renewTime;
  uint32_t rebindTime;
  IPV4_ADDR dhcpAddress;
  IPV4_ADDR serverAddress;
  const char* bootFileName;
} TCPIP_DHCP_INFO;
Members
Members 
Description 
TCPIP_DHCP_STATUS status; 
current status 
uint32_t dhcpTime; 
current DHCP time, seconds the following fields are significant only if a lease has been obtained and is currently valid i.e. status >= TCPIP_DHCP_BOUND 
uint32_t leaseStartTime; 
time when lease was requested 
uint32_t leaseDuration; 
lease duration as returned by the server, seconds 
uint32_t renewTime; 
the time for moving to renew state, seconds 
uint32_t rebindTime; 
the time for moving to rebind state, seconds 
IPV4_ADDR dhcpAddress; 
IPv4 address obtained by DHCP 
IPV4_ADDR serverAddress; 
IPv4 address of the server that granted the lease 
const char* bootFileName; 
pointer to the bootfile name that was returned by the server This will be 0 if TCPIP_DHCP_STORE_BOOT_FILE_NAME option is not enabled!
