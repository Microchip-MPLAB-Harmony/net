TCPIP_MAC_SynchReqF Type

Synchronization request Function: typedef bool (*TCPIP_MAC_SynchReqF)(void* synchHandle, TCPIP_MAC_SYNCH_REQUEST request); 

This function describes the MAC synchronization request function. The MAC driver will call this function whenever it needs to create, delete, lock or unlock a synchronization object.

C
typedef bool (* TCPIP_MAC_SynchReqF)(void* synchHandle, TCPIP_MAC_SYNCH_REQUEST request);
Parameters
Parameters 
Description 
synchHandle 
  • pointer to a valid storage area
  • for TCPIP_MAC_SYNCH_REQUEST_OBJ_CREATE it is an address that will store a handle to identify the synchronization object to the OS (OSAL).
  • for other synchronization object request types the handle has to be valid and identify the synchronization object to the OS (OSAL).
  • for Critical sections it maintains OS passed info
 
request 
a TCPIP_MAC_SYNCH_REQUEST type 
Returns

  • true - if the call is successful
  • false - if the call failed

Remarks

None.
