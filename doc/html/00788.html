Introduction 
TCP/IP Stack Library iperf Module for Microchip Devices
This library provides the API of the iperf module that is available on the Microchip family of devices with a convenient C language interface. It is a module that belongs to the TCP/IP Stack.
The module provides network performance measurement. 
Using Iperf 
This section describes how to use in the TCP/IP Stack Library Iperf module. 
Using the Library 
This topic describes the basic architecture of the Iperf TCP/IP Stack Library and provides information and examples on its use. 
Configuring the Module 
The configuration of the Iperf TCP/IP Stack is based on the template file iperf_config.h with the actual symbols generated in the configuration.h file.
This header file contains the configuration selection for the Iperf TCP/IP Stack. Based on the selections made, the Iperf TCP/IP Stack may support the selected features.
