Running the Demonstration

For SAM E70/V71 Xplained Ultra Evaluation Kit

  1. Configure the Development Board as given here.
  2. Make the demonstration setup as shown here.
  3. Build and download the demonstration project on the target board.
  4. A virtual COM port will be detected on the computer, when the USB cable is connected.
  5. Open a standard terminal application on the computer (like Hyper-terminal or Tera Term) and configure the virtual COM port.
  6. Set the serial baud rate to 115200 baud in the terminal application.
  7. Restart the demonstration by pressing the 'RESET' button on the board.
  8. See that the initialization prints on the serial port terminal.
  9. When the DHCP client is enabled in the demonstration, wait for the DHCP server to assign an IP address for the development board. This will be printed on the serial port terminal.
  10. The SNMP and SNMPv3 server is hosted by the demonstration application.
  11. Run tcpip_discoverer to get the IPv4 and IPv6 address for the board.
  12. Open a SNMP manager (iREASONING SNMP manager is recommended)
  13. Follow the steps given in iREASONING Networks MIB Browser section in the Third-Party help for complete details on using and configuring the application demonstration using the iREASONING SNMP Manager.

 

SNMP MIB Browser

Several SNMP MIB browsers are available. Users can also install a customized MIB browser specific to their application. 

SNMP Get, GetNext, GetBulk, Set request and response are working as expected for SNMP v1/v2/v3 versions.

  • For SNMP v2c , the Agent is configured with three Read communities ("public", "read", " ") and three Write communities ("private","write","public").
