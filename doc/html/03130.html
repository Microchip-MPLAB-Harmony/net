TCPIP_SMTPC_CLIENT_ADDR_BUFFER_SIZE Macro

size of a buffer that can hold an email address: user@domain.smth

C
#define TCPIP_SMTPC_CLIENT_ADDR_BUFFER_SIZE 80
Remarks

this buffer is automatic, created on the stack Check your project stack settings if you need a large buffer here! If buffer not long enough this could generate TCPIP_SMTPC_RES_MESSAGE_ADDR_LEN_ERROR.
