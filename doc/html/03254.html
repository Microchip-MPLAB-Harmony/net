TCPIP_SMTPC_SERVER_REPLY_TIMEOUT Macro

general server response timeout, seconds Should be adjusted according to the server responsivity RFC specifies it as 5 min Default is 1 minute, which should normally be enough

C
#define TCPIP_SMTPC_SERVER_REPLY_TIMEOUT 60
