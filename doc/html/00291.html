DDNS_SERVICES Enumeration

Dynamic DNS Services. Must support the DynDNS API (Auxlang) and correspond to ddnsServiceHosts and ddnsServicePorts in DynDNS.c.

C
typedef enum {
  DYNDNS_ORG = 0u,
  NO_IP_COM,
  DNSOMATIC_COM
} DDNS_SERVICES;
