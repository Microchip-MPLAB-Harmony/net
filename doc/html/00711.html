ICMP_ECHO_RESULT Enumeration

result of an ICMP Echo Request call

C
typedef enum {
  ICMP_ECHO_OK = 0,
  ICMP_ECHO_ALLOC_ERROR = -1,
  ICMP_ECHO_ROUTE_ERROR = -2,
  ICMP_ECHO_TRANSMIT_ERROR = -3,
  ICMP_ECHO_PARAMETER_ERROR = -4,
  ICMP_ECHO_BUSY = -5,
  ICMP_ECHO_BAD_HANDLE = -6
} ICMP_ECHO_RESULT;
Members
Members 
Description 
ICMP_ECHO_OK = 0 
operation successful error codes, < 0 
ICMP_ECHO_ALLOC_ERROR = -1 
could not allocate memory 
ICMP_ECHO_ROUTE_ERROR = -2 
could not find a route to destination 
ICMP_ECHO_TRANSMIT_ERROR = -3 
could not transmit (dead interface, etc.) 
ICMP_ECHO_PARAMETER_ERROR = -4 
null pointer, etc. 
ICMP_ECHO_BUSY = -5 
module currently busy 
ICMP_ECHO_BAD_HANDLE = -6 
invalid handle, no such request exist
