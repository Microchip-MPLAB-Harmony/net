TCPIP_UDP_ServerOpen Function

Provides a unified method for opening UDP server sockets.

C
UDP_SOCKET TCPIP_UDP_ServerOpen(
    IP_ADDRESS_TYPE addType, 
    UDP_PORT localPort, 
    IP_MULTI_ADDRESS* localAddress
);
Preconditions

UDP is initialized.

Parameters
Parameters 
Description 
IP_ADDRESS_TYPE addType 
The type of address being used. 
Returns

  • INVALID_SOCKET - No sockets of the specified type were available to be opened or parameter error.
 

  • A UDP_SOCKET handle - Save this handle and use it when calling all other UDP APIs.

Example

IP_ADDRESS_TYPE_IPV4 or IP_ADDRESS_TYPE_IPV6. It can be IP_ADDRESS_TYPE_ANY and the server socket will accept any incoming type of connection. 

UDP_PORT localPort - UDP port on which to listen for connections 

IP_MULTI_ADDRESS* localAddress - Pointer to a local IP address to use. This needs to be the valid IP address of a local interface the server is to listen on or NULL if any incoming interface will do.
