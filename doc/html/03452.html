TCPIP_STACK_HEAP_HighWatermark Function

The function returns the maximum amount of memory that's been allocated from the heap since the heap was created.

C
size_t TCPIP_STACK_HEAP_HighWatermark(
    TCPIP_STACK_HEAP_HANDLE heapH
);
Preconditions

heapH - valid heap handle

Parameters
Parameters 
Description 
heapH 
handle to a heap 
Returns

The maximum allocated memory from the heap.

Remarks

None.
