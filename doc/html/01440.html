Refer to each section for a detailed description.

Description

TCPIP_HTTP_NET_ConnectionPostExecute 
Processes POST form variables and data. 
 
TCPIP_HTTP_NET_ConnectionPostSmSet 
Set the POST state machine state. 
 
TCPIP_HTTP_NET_ConnectionRead

TCPIP_HTTP_NET_ConnectionStatusSet
