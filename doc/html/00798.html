IPV4_FRAGMENT_INFO Union

IPv4 packet fragment definition 

This is the structure of an IPv4 fragment info in the IPv4 header.

C
typedef union {
  uint16_t val;
  struct {
    uint16_t fragOffset : 13;
    uint16_t MF : 1;
    uint16_t DF : 1;
    uint16_t reserved : 1;
  }
} IPV4_FRAGMENT_INFO;
Members
