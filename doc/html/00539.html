DRV_GMAC_ParametersGet Function

MAC Parameter Get function TCPIP_MAC_RES DRV_GMAC_ParametersGet(DRV_HANDLE hMac, TCPIP_MAC_PARAMETERS* pMacParams); 

This is a function that returns the run time parameters of the MAC driver.

C
TCPIP_MAC_RES DRV_GMAC_ParametersGet(
    DRV_HANDLE hMac, 
    TCPIP_MAC_PARAMETERS* pMacParams
);
Preconditions

DRV_GMAC_Initialize() should have been called. DRV_GMAC_Open() should have been called to obtain a valid handle.

Returns

  • TCPIP_MAC_RES_OK if pMacParams updated properly
  • a TCPIP_MAC_RES error code if processing failed for some reason

Remarks

None.
