HTTP_FILE_TYPE Enumeration

File type definitions

C
typedef enum {
  HTTP_TXT = 0u,
  HTTP_HTM,
  HTTP_HTML,
  HTTP_CGI,
  HTTP_XML,
  HTTP_CSS,
  HTTP_GIF,
  HTTP_PNG,
  HTTP_JPG,
  HTTP_JS,
  HTTP_JAVA,
  HTTP_WAV,
  HTTP_UNKNOWN
} HTTP_FILE_TYPE;
Members
Members

HTTP_GIF

HTTP_PNG 
File is PNG image (extension .png) 
HTTP_JPG 
File is JPG image (extension .jpg) 
HTTP_JS 
File is java script (extension .js) 
HTTP_JAVA 
File is java (extension .class) 
HTTP_WAV 
File is audio (extension .wav) 
HTTP_UNKNOWN 
File type is unknown
