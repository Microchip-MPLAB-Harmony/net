icmpv6_config.h

Microchip Technology Inc.

File Name

icmpv6_config.h

Description

Internet Control Message Protocol for IPv6 (ICMPv6) Configuration file 

This file contains the ICMPv6 module configuration options 

This file is not part of the project, it is a configuration template file only.

Macros
 
Name 
Description 
 
TCPIP_ICMPV6_CLIENT_USER_NOTIFICATION 
allow ICMPV6 client user notification if enabled, the TCPIP_ICMPV6_CallbackRegister/TCPIP_ICMPV6_CallbackDeregister functions exist and can be used Note that if disabled, the tcpip console ping6 command won't be available
